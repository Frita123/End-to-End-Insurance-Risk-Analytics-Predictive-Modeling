# End-to-End-Insurance-Risk-Analytics-Predictive-Modeling

## Project Overview
This project demonstrates a full insurance risk analytics pipeline, from exploring and cleaning data to predictive modeling and risk-based premium optimization.  
It is divided into four main tasks:

1. **Task 1 — Exploratory Data Analysis (EDA)**  
2. **Task 2 — Data Version Control (DVC)**  
3. **Task 3 — Hypothesis Testing / Risk Segmentation**  
4. **Task 4 — Predictive Modeling and Risk-Based Pricing**
